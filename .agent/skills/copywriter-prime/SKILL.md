# Copywriter Prime — High-Agency Voice

Avoid "AI Slop" (words like *tapestry*, *leverage*, *unleash*, *comprehensive*, *transformative*). Use human-first language that is specific, punchy, and grounded in reality.

## 1. The "AIDA" Framework
Every landing page must follow this structure from top to bottom:
- **Attention (Hero)**: A headline that makes a specific, bold claim.
- **Interest (Value Props)**: Explain exactly *how* it works with evidence.
- **Desire (Social Proof/Manifesto)**: Show why life is better with the product.
- **Action (CTA)**: A single, clear, and high-contrast command.

## 2. Strict Ban List (Eradicate these words)
- Delve, Tapestry, Unleash, Embrace, Foster, Elevate, Comprehensive, Transformative, Integrated, Synergize, Pivot, Cutting-edge.
- **NEVER** use the phrase "Empowering you to..." or "The ultimate solution for...".

## 3. High-End Copy Patterns
- **The Contrast**: "Most [Industry] focuses on [Common/Bad approach]. We focus on [Differentiated/Good approach]."
- **The Specific Number**: "Reduces [Pain] by [X]%" instead of "Reduces [Pain] significantly."
- **The Short Sentence**: Break up long academic sentences. Use 10 words or less for headlines.

## 4. UI Copy Best Practices
- **Buttons**: Use "Start Building" instead of "Get Started". Use "See how it works" instead of "Learn More".
- **Inputs**: Use "Your best email" instead of "Email address".
- **Empty States**: Don't say "No items found." Say "Your dashboard is waiting for its first entry."
